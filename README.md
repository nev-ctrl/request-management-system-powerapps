# 📋 Request Management System — Power Platform

A low-code application built on Microsoft Power Platform to manage and track 
user requests through structured digital forms with automated workflows.

---

## 🚀 Features

- Submit and manage requests via structured digital forms
- Automated workflows using Power Automate
- Data stored and managed via Excel / Dataverse
- Analytics and reporting dashboard built in Power BI

---

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| Microsoft Power Apps (Canvas) | Frontend app & form UI |
| Microsoft Power Automate | Workflow automation & notifications |
| Microsoft Power BI | Reporting & analytics |
| Excel / Dataverse | Data storage & management |

## 📥 How to Import the App

1. Download the `.zip` export file from this repo
2. Go to [make.powerapps.com](https://make.powerapps.com)
3. Click **Apps** → **Import canvas app**
4. Upload the `.zip` and follow the prompts
5. Reconnect your data source if needed
